# BTC Signal v5 — Custom Timer

This version fixes the timer issue.

## New timer behavior

You can now enter the exact time remaining:

- Minutes left
- Seconds left

Then tap **Start Custom Timer**.

Examples:

- If Robinhood shows 10:52, enter:
  - Minutes left: 10
  - Seconds left: 52
  - Tap Start Custom Timer

- If Robinhood shows 7:08, enter:
  - Minutes left: 7
  - Seconds left: 8

There is also:
- Pause Timer
- Set 15:00
- Reset / Stop

## Everything else

Same optimized v4 logic:
- No API keys
- Live BTC polling from Coinbase, Kraken, CoinGecko
- Anti-flip filters
- WATCHING before entry
- Final 60-second average awareness
- Up/Down optional Robinhood prices
