Safety notes
This is a standalone manual decision aid. It does not require contract prices. It always produces a local evidence decision from live BTC venue data, target distance, timer, microstructure, active trade state, and session memory. The optional backend is a second opinion only; it cannot override valid local evidence with stale/schema blockers. It cannot guarantee market outcomes or see Robinhood’s licensed settlement feed.
