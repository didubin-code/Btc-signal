Integration Hook for your 15m tool
This page writes localStorage.btcPressureRadar and broadcasts through BroadcastChannel('btc-pressure-radar'). Your main tool can listen and veto ABOVE trades when score >= 65, or mark final-minute strike danger when level is DANGER or CRITICAL. 
const ch = new BroadcastChannel('btc-pressure-radar');
ch.onmessage = (ev) => {
  const p = ev.data;
  if (p.score >= 65 || p.level === 'CRITICAL') {
    // VETO ABOVE / warn of external sell-pressure threat
  }
};
